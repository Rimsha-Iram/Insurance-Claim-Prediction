# Insurance Claim Prediction

## Objective
Estimate medical insurance claim amounts using personal data.

## Approach
- Data Understanding & Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Linear Regression & Random Forest Models
- Model Evaluation using MAE and RMSE

## Results
- Linear Regression MAE: 0.27, RMSE: 0.42
- Random Forest Regressor MAE: 0.20, RMSE: 0.37

## Insights
- **Smoking** and **high BMI** are major cost drivers.
- **Age** positively influences medical costs.
- Gender, region, and number of children have lower impact.
- Random Forest provides better accuracy for outlier cases.
